## Setup

To run Notes App locally:
1. Clone the repository:
   
    ```
   git clone https://github.com/Volodymyr1233/Notes-App.git
    ```
2. Install dependencies:
    ```
    npm install
    ```
3. Run project:
    ```
    npm start
    ```
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.
